設備管理の現場が直面する課題

PROBLEM 01

PROBLEM 02

PROBLEM 03
センサー設置のたびに専門工事が必要で、コストと時間がかかりすぎる
PROBLEM 04

Why Robosensor

ロボセンサー®が選ばれる理由

ピエゾフィルムを芯線に巻き付けた独自構造により、従来センサーの常識を超えた取り付け自由度と高精度検出を両立しています。

POINT 01
外径約0.5mm — 業界最細水準のワイヤー型
細くしなやかなワイヤー形状で、曲面・狭小部・配管まわりへの設置が簡単。後付けも容易で、既存設備へ最小限の工事で導入できます。
POINT 02
電線自身がセンサー — 独自のセンシング設計
センサーケーブル自体が振動を直接検出する革新的な設計。接触面の振動を正確にセンシングし、従来センサーとは根本的に異なる取り付け自由度を実現。
POINT 03
広帯域・低ノイズ — 30Hz〜40KHzの高精度検出
広い周波数帯域で微細な振動変化も逃さず検出。初期異常の早期発見により、突発的なダウンタイムを大幅に削減します。
POINT 04
プリアンプ内蔵 — 接続するだけで即計測
センサーケーブルにプリアンプを内蔵。電源アンプ（ASU001A）にMicroUSBで接続するだけで、複雑な設定不要ですぐに振動計測を開始できます。
Lineup

2つのラインアップから選べる

設置環境や計測対象に合わせて、ストレート型・ループ型の2種類からお選びください。

SPS
ストレート型
直線状のワイヤーを配管・シャフト等の長手方向に沿わせて設置。終端部あり。直線的な設備・長尺部材の振動検出に最適です。
標準サイズ：センサー 30cm ＋ ケーブル 1m
SPL
ループ型
ループ状のワイヤーで円筒体・回転体を囲むように設置。回転軸・ポンプ・モーターなど、丸みのある設備の全周振動検出に最適です。
標準サイズ：センサー 30cm ＋ ケーブル 1m
Use Cases

さまざまな現場で活躍

製造業・インフラ・エネルギーなど、幅広い業種の設備異常検知に導入されています。

製造業
工作機械・CNCスピンドル
スピンドル外周や細径部にワイヤーを巻き付けて設置。切削時の異常振動を早期検出し、工具折損・加工不良を防止。
→ SPS をシャフトに沿わせるか SPL で軸を囲む
製造業
搬送ライン・コンベア
ローラー・チェーン駆動部の振動をリアルタイムで常時モニタリング。ベアリング劣化の兆候を早期検知し、計画外停止ゼロを実現。
→ ローラー軸受け部に SPL を巻き付けて設置
製造業
産業用ロボット関節部
アーム関節の微細な振動を検知し、減速機の摩耗・ギアバックラッシュ増大を早期把握。メンテナンスサイクルの最適化に貢献。
→ 関節周囲の狭小部にワイヤーを沿わせるだけ
インフラ・建設
配管・ポンプ設備
複雑な配管系に沿わせて設置し、異常振動・ウォーターハンマーを検知。漏水・破損の予兆を早期に察知します。
→ 配管外周に SPS を巻き付け、接着・結束で固定
インフラ・建設
橋梁・構造物モニタリング
橋梁や構造物の微細な振動・共振周波数の変化を長期モニタリング。損傷・劣化の進行を非破壊で把握し、維持管理に活用。
→ 曲面・鋼材に沿ってワイヤーを貼り付け
エネルギー
発電機・タービン
高温・高振動環境下の回転体に設置し、不平衡・軸ずれ・ブレード疲労亀裂を早期検出。稼働率の最大化に貢献します。
→ SPL で回転軸まわりを囲み固定
エネルギー
風力発電設備
ナセル内部の増速機・発電機軸受けに設置。常時モニタリングで高所作業を伴う点検コストを大幅に削減。
→ 軸受けハウジング周囲に沿わせて設置
食品・医薬
充填機・包装機械
クリーンルーム内でも、細くコンパクトなワイヤーで目立たず設置。シール不良・充填量ばらつきにつながる機械振動の変化を常時監視。
→ 駆動部に SPS を密着・固定するだけ
